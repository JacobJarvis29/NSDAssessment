~~Installation and Execution~~
Simply extract the zip folder and launch in IDE of choice.
The system will run on your localhost with a port of 12345 so make sure that is clear.
Run the server executable first, then launch a client executable.
~~Commands~~
To access the system, you must first login or you will be denied
login [username] [password]
post [message] [channel]
read
sub [channel]
unsub [channel]
find [message]
quit

No extra packages needed.
